All About Me

“One can furnish a room very luxuriously by taking out furniture rather than putting it in.” - Francis Jourdain

To Me To You

I love to collaborate on projects, not only does it make creating a project a lot smoother but it also means that ideas are nurtured and transformed with the 'everyday' user. After all, it's easier to get an amazing user experience if it's the user that's helps plan out the design.

A Scribble Here...

I prefer to sketch any wireframes before I get into the nitty gritty interface designs on Photoshop. Generally, my interface designs focus on concise content, intuitive UI patterns and simple interactions. Essentially, I'm a minimalist who truly believes that less is more.

Code Me Up Scotty!

Being a minimalist I like to design in the browser using HTML5, CSS3 and a touch of JavaScript. I love coding things from scratch, but I can work with front-end frameworks like Bootstrap and can add Content Management Systems like Wordpress into the mix.
