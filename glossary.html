The Mumbo Jumbo

Words, words and more words

#

404

A page the user sees when they try to reach a non-existent page on a website.

A

Accessibility

The ability for a website to be used by everyone regardless of their disabilities i.e. visually impaired visitors using screen readers.

.ai

A vector file format used for Adobe Illustrator.

ALT Attribute

Alternate text used to describe an image. Used by search engines and screen readers in place of an image.

Anchor

The part of a webpage where a link begins or ends.

B

Branding

A brand includes all the messages, promotions, products etc. that a particular business uses.

Breakpoints

The points at which a website’s content will adjust to provide the user with the best possible layout to view content. Breakpoints are often defined by device widths that are being targeted.

C

CMS

(Content Management System)
This system allows a user to edit, delete and manage the content of a website from a visual interface.

Cookies

A small text file that contains an anonymous unique identifier and visit information that is sent to a browser from a website and is then stored on a visitor’s computer hard drive. This data can provide information about who visits the website, how often they visit and their preferences.

CSS

(Cascading Style Sheets)
A file that contains the colours, fonts, layout and overall appearance of a website. A style sheet may only need to be altered once to have an effect on multiple pages of a website, hence having a cascading effect.

CTA

(Call to action)
A button, arrow or other graphic element, which entices a user to perform a particular action.

D

DOCTYPE

The doctype declaration specifies which version of HTML is used in a document.

Domain

The name of the website that a user enters into a browser to visit it. For example, joeshipton.co.uk

Drop-Down

A drop-down can be one of two elements: A drop-down menu is a navigational menu that has sub-menus. A drop-down field is found within a form on a website.

E

Element

A HTML element is everything from the start tag to the end tag. For example <p></p>for paragraphs.

.eps

(Encapsulated PostScript)
A common vector file format which allows a graphic to be resized without altering the quality or resolution.

F

Favicon

An icon image that is associated with a specific website. The favicon is displayed in either the title bar or tab of the browser.

Flyout Navigation

A flyout navigation menu contains multiple links to different pages. It’s typically contains multiple columns, whereas a drop-down menu may only display one column below the main tab.

FTP Client

The software that is used to upload a website to a host server.

G

.gif

(Graphics Interchange Format)
A simple graphic, which has a can be compressed to minimal sizes. Typically a low quality image which also supports animation.

H

Hero

A common term used to refer to the main image on a homepage or landing page.

Hexadecimal

A base-16 numbering system used to define colours online.

Hosting

The web servers where your website files are housed, served, and maintained.

HTML

(HyperText Markup Language)
Coding language for web pages. Essentially combined tags to create the structure of any website. The tags are put together in pairs with the content of a website filling in the spaces in between.

HTTP

(HyperText Transfer Protocol)
This is a method used to transfer information on the internet.

HTTPS

(HyperText Transfer Protocol over SSL (Secure Socket Layer))
Like HTTP, it’s a method to transfer information on the internet but it’s done over a secure encrypted connection.

Hyperlink

A hyperlink is a link from one web page to another, either on the same site or another one.

I

Input field

An input field is a blank area where a user would input specific text based information. They are found within a form on a website.

J

JavaScript

A lightweight, interpreted, programming language which is the generally used as the scripting language for web pages.

.jpg

A JPEG image is created with lose compression which means parts of the image's data is removed to create a smaller file size. This makes the file easily viewable on the web and is the most common file type.

jQuery

The most popular JavaScript library that can add extra functionality to their websites.

L

Landing page

A page on a website that is generally used for direct promotional content which is often presented to a user on page load or from a marketing link.

Less

A dynamic style sheet language that can be compiled into CSS.

M

Meta Tag

An HTML tag that stores information about a web page, such as description, author, copyright, etc. Their function is to provide information about a web page and its content to search engines.

Mock-Up

Design for the final webpage.

O

Optimization

The reduction in file size of a linked asset so it's more suitable for use on the web.

P

PHP

(HyperText Preprocessor)
PHP allows for dynamic web content to be displayed and interacted with on a web page.

.png

(Portable Network Graphics)
Another image file that is to a GIF. The difference is that a PNG allows for transparency and can’t be used in a CMYK (print) colour space. It’s meant for use on the web. PNG does not support animation.

Preprocessor

A program that lets you generate a web language from the preprocessor's own unique syntax. Features of preprocessors make the CSS structure more readable and easier to maintain. See Less, Sass, jQuery.

.psd

Adobe Photoshop file format
One of the main tools used for photography editing, website designs and various other graphics.

R

Responsive Design

A website that adjusts to the screen it is being viewed on, whether desktop, mobile or smart phone. Media queries are used to adjust the styles based on the resolution of the device.

Rollover

An action that happens when a user places the cursor over a particular element that has a rollover effect applied to it.

S

Sass

A dynamic style sheet language that can be compiled into CSS.

SEO

(Search Engine Optimization)
Planning out and writing and the page's content to improve the likelihood of it showing up when relevant search terms are used in search engines such as: Google, Yahoo and Bing.

Style guide

A document that includes all of the colours, fonts, and branding guidelines for a website, brochure, book or complete brand.

.svg

(Scalable Vector Graphics)
a vector graphics file format that enables two-dimensional images to be displayed in XML pages on the Web. Vector images are created through text-based commands formatted to comply with XML specifications.

T

Title Attribute

Text used to describe the webpage in the top of the browser window and also used on images to describe the image on hover.

U

UI

(User Interface)
A focus on the usability of a website. Makes sure the targeted user can use the website with ease by designing elements to make their experience easy.

UX

(User Experience)
Refers to how a user feels about a product or a website. Linked to User Interface.

W

White Space

Refers to the space around elements. White space allows the elements of a web page to stand out to the user.

Wireframe

Very simplistic drawings and layouts of a webpage. These typically have no design applied. Wireframes are used in the preliminary stages of a website design.

WYSIWYG

(What You See Is What You Get)
It’s the interface inside a CMS that applies styles to text and graphics and allows the user to see what the content will look like.
